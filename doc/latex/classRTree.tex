\doxysubsection{Detailed Description}
Class representing an R-\/tree. 

An R-\/tree is a tree data structure used for efficient search and insertion of 2D spatial data. It maintains a balanced tree of nodes, each of which has a bounding box covering all of its children. A node can be either a leaf or an internal node. Leaf nodes store triangles, while internal nodes store pointers to other nodes.
