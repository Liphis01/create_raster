\doxysubsection{Detailed Description}
Struct representing a node in an R-\/tree. 

A node in an R-\/tree can be either a leaf or an internal node. If it is a leaf, it stores a triangle, otherwise it stores a list of children. Each node also has a bounding box that covers all of its children.
